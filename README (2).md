
# BYTE ECOMMERCE

It is an e-commerce application developed using the MERN stack, Tailwind CSS for styling, and Redux Toolkit for seamless state management. This modern platform caters to all essential e-commerce functionalities such as user authentication, dynamic filtering and search, an intuitive admin dashboard, product reviews, and user profile management. Leveraging various React packages, the application ensures a robust and feature-rich experience for both customers and administrators.

## Features

- Single-page Web Application 
- Fully responsive
- Admin dashboard
- Products management
- User profile management


## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express, Mongodb

## Environment Variables

To run this project, you will need to add the following environment variables to your config.env file in root backend folder.

`PORT`

`DB_URL`

`JWT_SECRET`

`JWT_EXPIRE`

`COOKIE_EXPIRE`

`SMPT_SERVICE`

`SMPT_MAIL`

`SMPT_PASSWORD`

`CLOUDINARY_NAME`

`CLOUDINARY_API_KEY`

`CLOUDINARY_API_SECRET`



## Run Locally

Clone the project

```bash
  git clone https://github.com/Nikeshchaudhary52494/ecommerce_webSite.git
```

Go to the project directory

```bash
  cd ecommerce_webSite
```

Install frontend dependencies

```bash
  cd frontend
```
```bash
  npm i --legacy-peer-deps
```
Install backend dependencies

```bash
  cd backend
```
```bash
  npm i
```

Start frontend server

```bash
  npm start
```
Start backend server

```bash
  npm run dev
```
